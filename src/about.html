📌 项目简介

这是一个基于 Vite 构建、托管在 Cloudflare Pages 上的纯静态个人站点，采用无服务器架构，零运维成本，代码推送自动部署。全站采用玻璃拟态设计风格，支持明暗双主题一键切换。

🛠️ 技术栈
HTML5 CSS3 原生 JavaScript Vite Cloudflare Pages Pages Functions
📁 功能模块
  • 链接收藏舱：分类管理常用网址，下拉选择一键跳转
  • 图片图库：响应式瀑布流展示，点击查看大图
  • 订阅工具导航：节点订阅面板快速入口
  • API 测试接口：基于 Pages Functions 的轻量边缘 API 演示
⚡ 部署特性
  • 全球边缘节点加速，国内访问速度优秀
  • Git 推送自动触发构建部署，无需手动操作
  • 零服务器运维，免费额度满足个人使用
  • 支持自定义域名、HTTPS 自动证书
